# 📌 CHANGELOG

Todas las modificaciones importantes del proyecto se documentan en este archivo.

## [v1.0] - Primera versión estable

🟢 Funcionalidades principales:
- Búsqueda de pisos en la costa de Galicia (obra nueva y reformas).
- Filtro por terraza mínima, vistas al mar y precio.
- Envío diario automático a Telegram.
- Generación de resumen en CSV.
- Separación segura de configuración (`config_local.json` excluido del repo).
- Plantilla de configuración incluida (`config_example.json`).
- Script auxiliar para obtener `chat_id` de Telegram (`get_chat_id.py`).

🔐 Seguridad:
- `.gitignore` actualizado para proteger claves sensibles.
- README con notas de seguridad añadidas.
